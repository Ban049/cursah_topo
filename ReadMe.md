#  Руководство пользователя и администратора (NoteServiceForYou)

Приложение представляет собой консоль для управления заметками, пользователями и мониторингом аппаратных ресурсов системы.

---

##  1. Авторизация и Сессии
Приложение поддерживает автоматическое восстановление сессии при перезапуске.
*   **`login -u [имя] -p [пароль]`** — Вход в систему. 
```bash
 login -u name -p pass111
```
*   **`logout`** — Выход из текущей сессии и удаление локального токена.
```bash
logout
```
---

##  2. Заметки (Доступно для роли: пользователь)
Пользователи имеют доступ к полному CRUD-циклу, но **только для собственных** заметок.
*   **`note list`** — Вывод списка всех ваших заметок.
```bash
note list
```
*   **`note add -t "Заголовок" -c "Содержимое"`** — Создание новой заметки.
```bash
note add -t "Заметка" -c "Важное"
```
*   **`note edit -id [ID] -t "Новый заголовок" -c "Новый текст"`** — Редактирование существующей заметки.
```bash
note edit -id 4 -t "Изменено" -c "Новое содержание"
```
*   **`note del -id [ID]`** — Удаление заметки.
```bash
note del -id 4
```

---

##  3. Управление пользователями (Доступно для роли: администратор)
Администратор управляет доступом в систему. У него **нет** прав на чтение чужих заметок или запуск мониторинга.
*   **`user list`** — Список всех пользователей системы и их статусов (Активен / Заблокирован).
```bash
user list
```
*   **`user add -u [имя] -p [пароль] -r [Роль]`** — Создание пользователя.
    *   *Доступные роли:* `User`, `Admin`, `WatchdogAdmin`.
```bash
user add -u Test -p 12345 -r User
```	
*   **`user block -id [ID] -d [дни]`** — Блокировка учетной записи пользователя на указанное количество дней.
```bash
user block -id 10 -d 1
```
*   **`user del -id [ID]`** — Полное удаление пользователя из БД.
```bash
user del -id 6
```

---

##  4. Мониторинг "Watchdog" (Доступно для роли: WatchdogAdmin)
Модуль работает в отдельном фоновом потоке. Он собирает метрики ОС Windows (CPU, RAM, HDD).
*   **`watchdog run`** — Запуск фонового сбора аппаратных метрик.
```bash
watchdog run
```
*   **`watchdog stop`** — Остановка сбора метрик.
```bash
watchdog stop
```
*   **`watchdog config -i [секунды]`** — Изменение интервала сбора данных (по умолчанию 5 секунд).
```bash
watchdog config -i 10
```
*   **`watchdog status`** — Проверка текущего состояния фонового модуля.
```bash
watchdog status
```
*  	**`watchdog list -n [кол-во]`** — Просмотр последних собранных метрик из базы данных (например, `-n 10`).
```bash
watchdog list -n 10
```

##  5. Системные команды и Обновление (Доступно для роли: администратор)
Система поддерживает версионирование и автоматическое накатывание обновлений через внешний `Launcher.exe`.
*   **`system logs -n [кол-во]`** — Просмотр системного журнала ошибок и действий.
```bash
system logs -n 10
```
*   **`system update`** — Запуск процедуры подготовки системы к обновлению. 
    * *Программа автоматически найдет последнюю версию, создаст новую папку (например, `v-1.0.1`) и скопирует туда конфигурационный файл `appsettings.json`.*
```bash
system update
```	

###  Инструкция по обновлению через Лаунчер:
1. Зайдите в программу под ролью `Admin` и введите `system update`.
2. Скомпилируйте новую версию основного приложения (`Cursah.exe`).
3. Скопируйте свежий `.exe` в новую папку, которую создала программа.
4. Выйдите из приложения командой `exit`.
5. Запустите **NoteServiceForYou.exe**. Он обнаружит новую версию, покажет таймер обновления и автоматически запустит новую версию программы!
